Améliorez votre productivité avec l'IA dans Word

WordMagician est un module complémentaire pour Microsoft Word qui intègre la puissance de l'IA directement dans votre traitement de texte.

Il vous permet de générer du texte, améliorer votre style, traduire du contenu, résumer des documents et bien plus, directement depuis l'interface de Word.

Compatible avec plusieurs API d'IA gratuites (Gemini, Together, OpenRouter) pour vous donner le choix de l'IA qui vous convient le mieux.

Interface entièrement en français et intuitive, conçue pour s'intégrer parfaitement dans votre flux de travail.

Commencez en quelques clics

Télécharger manifest.xml

Fonctionnalités principales

WordMagician offre une variété d'outils pour améliorer votre rédaction :

Génération de texte

Créez du contenu original basé sur vos instructions et le contexte de votre document.

Traduction

Traduisez vos textes en plus de 40 langues différentes sans quitter Word.

Résumé

Obtenez un résumé concis de votre texte sélectionné pour gagner du temps.

Amélioration du style

Perfectionnez la clarté et la qualité de vos écrits avec des suggestions d'amélioration.

Correction grammaticale

Identifiez et corrigez les erreurs grammaticales pour des textes impeccables.

Rédaction académique

Transformez votre texte en style académique professionnel pour vos travaux universitaires.

Guide d'installation

Suivez ces étapes détaillées pour installer WordMagician dans votre Microsoft Word

1

Télécharger et extraire le ZIP

Téléchargez le fichier manifest.zip et extrayez son contenu dans un dossier sur votre disque local C

2

Accéder aux propriétés du dossier

Faites un clic droit sur le dossier extrait dans le Disque Local C et sélectionnez "Propriétés" dans le menu contextuel, puis accédez à l'onglet "Partage" et cliquez sur le premier bouton "Partager"

3

Sélectionner le compte utilisateur

Dans le menu déroulant, sélectionnez votre compte utilisateur Windows actif, puis cliquez sur le bouton "Partager" en bas de la fenêtre

4

Désactiver la découverte du réseau

Sélectionnez la première option dans la fenêtre pour désactiver la découverte du réseau et le partage des fichiers pour les réseaux publics

5

Copier le lien partagé

Cliquez sur le bouton "Copier" pour copier le lien du fichier partagé

6

Vérifier le lien partagé

Si tout a été fait correctement, vous devriez voir le lien du dossier partagé comme indiqué sur l'image. Si c'est le cas, vous pouvez fermer cette fenêtre.

7

Ouvrir les Options de Word

Ouvrez Microsoft Word et accédez aux Options comme indiqué sur l'image

8

Accéder au Centre de gestion de la confidentialité

Une fois les options Word ouvertes, cliquez sur le dernier onglet en bas "Centre de gestion de la confidentialité", puis sur "Paramètres du centre de gestion de la confidentialité" comme indiqué par les chiffres 1 et 2 sur l'image

9

Configurer les catalogues de confiance

Une nouvelle fenêtre s'ouvre. Suivez simplement les numéros indiqués sur l'image. Au niveau du numéro 2, collez le lien que vous avez copié à l'étape 5. Il est crucial que le lien ait exactement le même format et la même disposition que sur l'image.

10

Sélectionner le catalogue et afficher dans le menu

Dans la liste des catalogues en bas, sélectionnez celui que vous venez d'ajouter et cochez la case "Afficher dans le Menu" comme indiqué sur l'image. Appuyez sur OK et fermez proprement toutes les fenêtres ! Il est crucial de fermer complètement Microsoft Word et de le rouvrir pour que nos configurations prennent effet !

11

Accéder aux compléments dans Word

Dans Microsoft Word, allez dans l'onglet "Insertion", puis cliquez sur "Mes compléments" et sélectionnez "Afficher tout" pour accéder à la gestion des compléments.

12

Sélectionner l'add-in dans le dossier partagé

Dans la fenêtre des compléments, naviguez vers l'onglet "Dossier partagé". Vous y trouverez l'add-in WordMagician. Cliquez sur l'add-in, puis sur le bouton "Ajouter" en bas de la fenêtre pour l'installer.

13

Utiliser l'add-in WordMagician

L'add-in WordMagician apparaît dans le ruban. Cliquez sur son icône pour commencer à l'utiliser

Note : Vous aurez besoin de configurer au moins une clé API dans les paramètres de l'add-in après l'installation. Vérifiez que le partage de dossiers est activé sur votre ordinateur.

Guide d'utilisation

Découvrez comment utiliser efficacement WordMagician dans Microsoft Word

Interface principale

Champ "Personnage"

C'est le ton ou le style que l'IA va adopter (professeur, rédacteur créatif, etc.)

Vous pouvez sélectionner un personnage prédéfini ou créer les vôtres

Champ "Instruction"

C'est la consigne précise que vous donnez à l'IA

Exemples : "Rédige une introduction sur...", "Explique ce concept..."

Langue de réponse

Sélectionnez la langue dans laquelle l'IA doit vous répondre

Boutons d'action

  • Traduire : traduit le texte sélectionné dans la langue choisie
  • Résumer : crée un résumé concis du texte sélectionné
  • Améliorer : améliore la clarté et le style du texte
  • Grammaire : corrige les erreurs grammaticales
  • Académique : transforme le texte en style académique
  • Paramètres : accède aux paramètres de l'add-in

Configuration API

Pour utiliser l'add-in, configurez au moins une API :

  1. Cliquez sur le bouton "Paramètres"
  2. Sélectionnez l'API souhaitée (Gemini, Together, OpenRouter)
  3. Entrez votre clé API et sélectionnez le modèle
  4. Cliquez sur "Confirmer" pour sauvegarder

Conseil : Les clés API sont stockées localement sur votre ordinateur pour plus de sécurité

Questions fréquentes

Trouvez rapidement des réponses aux questions les plus courantes

L'add-in est-il gratuit ?

L'add-in lui-même est gratuit, mais vous devez avoir vos propres clés API pour les services d'IA (Gemini, Together, OpenRouter). Ces clés API sont également gratuites à obtenir.

Où puis-je obtenir une clé API ?

Vous pouvez obtenir une clé API Gemini gratuitement sur le site de Google AI, une clé Together sur together.ai ou une clé OpenRouter sur openrouter.ai.

Mes données sont-elles sécurisées ?

Vos textes sont envoyés directement aux API d'IA que vous avez configurées. Vos clés API sont stockées localement dans votre navigateur et ne sont jamais partagées.

Puis-je utiliser l'add-in hors ligne ?

Non, une connexion internet est nécessaire car l'add-in se connecte à des API en ligne.

Quelle API est recommandée pour commencer ?

L'API Gemini de Google est recommandée pour débuter car elle offre une généreuse utilisation gratuite.

Comment changer la langue de l'interface ?

L'interface est en français par défaut. Vous pouvez changer la langue dans les paramètres en cliquant sur "Paramètres" puis en modifiant "Langue de l'interface".
